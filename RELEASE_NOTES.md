# Release Notes

## What's New

- Initial release of BDO-Font-Patcher for .NET 8.0

## Features

- Select any TTF font file
- Auto-detect Black Desert Online installation folder
- One-click font patching
- Dark theme UI
- Standalone EXE — no dependencies required

## Instructions

1. Download the `.exe` file from Assets below
2. Run it on any Windows 10/11 machine
3. Select your BDO game folder
4. Select a `.ttf` font file
5. Click "Use selected font"
6. Restart BDO
